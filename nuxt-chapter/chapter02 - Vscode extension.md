# Vsocode extension

VS Code Extensions សំខាន់ៗសម្រាប់ Nuxt

1. **Vue Language Features** (Volar)
    - Extension ផ្លូវការសម្រាប់ Vue & Nuxt
2. **TypeScript Vue Plugin** (Volar)
    - បង្កើន IntelliSense សម្រាប់ .vue files
    - ត្រូវ install ជាមួយ Volar
3. **Nuxt DevTools** (optional)
    - Debug Nuxt app ដោយផ្ទាល់
    - Inspect routes, components, state
4. **ESLint**
    - ពិនិត្យ code style & error
    - ធ្វើឲ្យ code ស្អាត និង standard
5. **Prettier** – Code formatter
    - Auto format code (Vue, JS, TS, JSON)
    - ធ្វើការជាមួយ ESLint បានល្អ
6. **Tailwind CSS IntelliSense** (បើប្រើ Tailwind)
    - Auto-complete class
    - Preview styles
7. **Path Intellisense**
    - Auto-complete file paths
    - មានប្រយោជន៍សម្រាប់ components, assets
